	Readme File for Code Example:
--------------------------------------------------

PIC32_Ports\blinky_leds

--------------------------------------------------
Code Example Description:
--------------------------------------------------
Example toggles LED lights on and off. Explorer16 LEDs are on lower 8-bits of PORTA and to use all LEDs, JTAG port must be disabled.
